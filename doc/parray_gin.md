parray_gin
==============

Installing
----------

    CREATE EXTENSION parray_gin;

Extension is compatible witgh PostgreSQL 9.1 and 9.2.

Description
-----------

[PostgreSQL](http://www.postgresql.org/) native extension providing GIN index support for arrays with partial match.
